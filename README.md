# 🎧 Musify - User Music Streaming Interface

## 📌 Overview
**Musify User Interface** is the frontend application for the Musify music streaming platform.  
It allows users to browse albums, explore songs, and enjoy a modern music streaming experience through a responsive and interactive interface.

The application communicates with a backend API to fetch music data such as albums, songs, and artist information.

This project focuses on building a **modern music streaming UI using React and modern frontend tools**.

---

## 🚀 Features

- 🎵 Browse and explore music albums
- 🎧 Play and manage songs
- 🔍 Search music and albums
- 📱 Responsive user interface
- ⚡ Fast frontend powered by Vite
- 🔗 Integration with backend REST APIs
- 🎨 Modern UI design with Tailwind CSS

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/RajatPandey10/Musify_Frontend_Users.git
```

Navigate to the project directory:

```bash
cd Musify_Frontend_Users
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Usage

Run the development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:5173
```

Make sure the backend server is running.
